# toeictester

### Chi tiết

* **Nodejs, Express**
* **Angular 4**
* **MongoDB**

### Cài đặt node module
```
npm i
```
### Chạy server

```
node app
```
### Chạy client 

```
npm start
```
### Lưu ý 
* *server và client không cần thiết phải chạy cùng nhau*
